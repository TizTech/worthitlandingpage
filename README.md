# Worth It? Landing Page

Static marketing site for the Worth It? app.

## Run locally

Open `worthitlanding/index.html` directly in a browser, or run a local static server.

Example:

```bash
cd worthitlanding
python3 -m http.server 8080
```

Then open `http://localhost:8080`.

## Notes

- No backend/auth required.
- App Store links are placeholder URLs.
- Fully responsive sections included:
  - hero
  - features
  - how it works
  - preview cards
  - privacy
  - CTA footer
